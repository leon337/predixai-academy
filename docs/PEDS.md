# PEDS — PredixAI Education Design System

## 1. Objetivo

Definir a linguagem visual e os padrões de interface da PredixAI Academy com foco em clareza, leitura prolongada, consistência e aprendizagem.

## 2. Filosofia visual

```text
Editorial
+ Tecnológica
+ Minimalista
+ Futurista
+ Profissional
```

A interface deve parecer um livro premium dentro de um software moderno.

## 3. Paleta

### Base

- `#07111F` — fundo escuro principal;
- `#0F1B2D` — superfície escura;
- `#16243A` — superfície elevada;
- `#F7F9FC` — fundo claro;
- `#FFFFFF` — branco.

### Marca e função

- `#2563EB` — ação e navegação;
- `#38BDF8` — destaque informativo;
- `#10B981` — progresso e conclusão;
- `#8B5CF6` — conteúdo avançado;
- `#F59E0B` — atenção;
- `#EF4444` — erro.

### Texto

- `#0F172A` — texto principal claro;
- `#64748B` — texto secundário claro;
- `#F8FAFC` — texto principal escuro;
- `#CBD5E1` — texto secundário escuro.

## 4. Tipografia

Princípios:

- leitura confortável;
- hierarquia curta e previsível;
- títulos claros;
- largura de linha controlada;
- fonte monoespaçada para código;
- uma ideia principal por seção.

Níveis:

1. Curso;
2. Módulo;
3. Capítulo;
4. Seção;
5. Corpo;
6. Legenda e observação;
7. Código.

## 5. Grid e espaçamento

Estrutura de desktop:

```text
Índice lateral | Conteúdo principal | Painel de apoio
```

Tablet:

```text
Conteúdo principal + painéis recolhidos
```

Celular:

```text
Conteúdo principal + navegação e painéis deslizantes
```

Escala de espaçamento:

```text
XS → S → M → L → XL → 2XL → 3XL
```

## 10. Ilustrações

Devem ser:

- simples;
- vetoriais;
- consistentes;
- com poucas cores;
- pedagógicas;
- reutilizáveis.

Ilustrações decorativas sem função de aprendizagem não entram no produto.

## 11. Motion

Permitido para:

- orientação;
- feedback;
- progresso.

Não pode atrasar, distrair ou bloquear a leitura.

## 12. Acessibilidade

Requisitos mínimos:

- contraste adequado;
- navegação por teclado;
- foco visível;
- ordem lógica;
- imagens com descrição;
- diagramas acompanhados de texto;
- vídeos com legenda;
- conteúdo preservado em desktop, tablet e celular.

## 13. Regra de implementação

Todo componente visual deve ser reutilizável, documentado e vinculado a uma necessidade real de interface ou aprendizagem.
